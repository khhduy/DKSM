Page Structure



  • index.html is for Home page one
  • index-2.html is for Home page two
  • about.html is for About page
  • cart.html is for Cart page
  • checkout.html is for Checkout page
  • compare.html is for Compare page between the Products
  • wishlist.html is for Wishlist page for the selected Products
  • my-account.html is for User dashboard page
  • login.html is for User log-in and register new account page
  • shop-1.html is for Default Shop Page (with left sidebar)
  • shop-4-grid.html is for 4grid Shop Page (without sidebar)
  • shop-5-grid.html is for 5grid Shop Page (without sidebar)
  • shop-grid-left-sidebar.html is for 4grid Shop Page (with left sidebar)
  • shop-grid-right-sidebar.html is for 5grid Shop Page (with right sidebar)
  • shop-list.html is for List Shop Page (without sidebar)
  • shop-list-left-sidebar.html is for List Shop Page (with left sidebar)
  • shop-list-right-sidebar.html is for List Shop Page (with right sidebar)
  • single-1.html is for Single Product Page of datails (normal view)
  • single-variable.html is for Single Product Page of datails (normal view with multiple select options)
  • single-left-tab.html is for Single Product Page of datails with vertical tab(leftside tab)
  • single-right-tab.html is for Single Product Page of datails with vertical tab(rightside tab )
  • single-slider.html is for Single Product Page of datails with full slider
  • single-gallery-left.html is for Single Product Page of datails with Gallery (leftside gallery)
  • single-gallery-right.html is for Single Product Page of datails with Gallery (rightside gallery)
  • blog-grid-left-sidebar.html is for Blog Grid Page (left sidebar)
  • blog-grid-right-sidebar.html is for Blog Grid Page (right sidebar)
  • blog-list-left-sidebar.html is for Blog List Page (left sidebar)
  • blog-list-right-sidebar.html is for Blog List Page (right sidebar)
  • blog-single-left-sidebar.html is for Single Blog Page (left sidebar)
  • blog-single-right-sidebar.html is for Single Blog Page (right sidebar)
  • contact.html is for Contact page
